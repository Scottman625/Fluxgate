## 📞 支援

如果您在使用過程中遇到問題：

1. **查看文檔**: 首先查看 [故障排除](./troubleshooting.md) 文檔
2. **搜尋問題**: 在 GitHub Issues 中搜尋相似問題
3. **提交問題**: 如果沒有找到解決方案，請提交新的 Issue

### 提交問題時請提供：
- 問題描述和重現步驟
- 環境資訊（作業系統、版本等）
- 錯誤日誌和堆疊追蹤
- 相關配置檔案（注意隱藏敏感資訊）
